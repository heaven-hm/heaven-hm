# Hey, I'm Heaven-HM 👋

I'm a game developer and reverse engineer focused on **Project I.G.I. (I'm Going In)** modding, scripting, and engine tooling.

## 👨‍💻 What I do
- 🛠️ Developing the **Project I.G.I. 3D Editor** — a modern, fully-featured 3D viewport level editor.
- 📦 Building **project-igi-converter** — a C++ Qt toolchain to compile, decompile, and export game scripts (QVM/QSC) and levels.
- 🔍 Reverse engineering I.G.I. file formats, task trees, schemas, and shaders.
- 🔗 Collaborating with the community to preserve and modernize classic game assets.

## 🎯 Current Projects
- **[project-igi-editor](https://github.com/heaven-hm/project-igi-editor)**: The core 3D level navigation and editor application.
- **[project-igi-converter](https://github.com/heaven-hm/project-igi-converter)**: Multi-format asset conversion engine and scripting editor.
- **[project-igi-qvm-editor](https://github.com/heaven-hm/project-igi-qvm-editor)**: Legacy script manipulation toolkit (Archived in favor of
